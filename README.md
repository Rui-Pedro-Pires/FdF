To download the files ↙️
```bash
git clone git@github.com:RuiPires999/fdf.git
```

To compile the program ↙️

```bash
$ make
```

To launch the program with a valid map as parameter
```bash
$ ./fdf ./maps/test_maps/42.fdf
```
